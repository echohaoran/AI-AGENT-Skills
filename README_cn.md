### [local-ocr](sources/local-ocr/SKILL_cn.md)
🔍 **本地 OCR 识图转文**，基于 Tesseract 引擎实现离线文字识别，无需网络 API，支持中英简繁四语，保护隐私同时离线可用。**主公系统已具备完整环境（Tesseract 5.5.2 + chi_sim/chi_tra/eng/jpn/kor语言包）**。

[查看详情](sources/local-ocr/SKILL_cn.md)

**🚀 即刻体验（零配置启动）：**
```bash
# 进入技能目录并运行测试脚本
cd ai-agent-commonly-used---kills/sources/local-ocr
python3 ocr_test.py

# 指定图片识别
tesseract your_image.png stdout -l chi_sim --psm 6
```

**✨ 核心能力：**

- ✅ **完全离线**：无需网络、无需 API 密钥、无隐私泄露风险
- ✅ **多语言支持**：简繁中日韩 + 80+ 种语言
- ✅ **批量处理**：一键处理文件夹内所有图片
- ✅ **Python 自动化**：提供封装好的 Python 函数直接调用
