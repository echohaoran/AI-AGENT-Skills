# 一键安装所有技能 — 提示词

> 共 **~82 项技能**，来自 [Gitee 归档仓库](https://gitee.com/echohaoran/ai-agent-skills)

---

## 📋 安装前提

1. Alma Desktop 已安装并运行
2. GitHub 网络可达（部分技能从 GitHub 拉取）
3. Gitee 网络可达（主归档仓库）

---

## 🚀 一键安装提示词

将以下整段文字粘贴给 Alma：

```
请帮我从仓库 https://gitee.com/echohaoran/ai-agent-skills 安装以下所有技能。

## 第1步：克隆归档仓库
```bash
cd ~/.config/alma/skills
git clone --depth 1 https://gitee.com/echohaoran/ai-agent-skills.git archive-repo
```

## 第2步：从归档仓库安装独立技能（28项）
以下技能路径为 archive-repo/sources/<技能名>，使用 alma skill install 从本地路径安装：

### 文档处理类
- awesome-privacy-skill  — 敏感数据脱敏与可逆还原
- document-xlsx   — Excel 文档处理
- office          — Office 文档综合处理
- pdf-generator   — PDF 生成引擎
- markitdown      — 微软文档转 Markdown（DOCX/Excel/PPTX/PDF/图片OCR）
- markdown-documentation — Markdown 文档风格检查
- local-ocr       — 本地 OCR 识图转文，离线可用
- Nano_Pdf        — PDF 轻量处理

### 工具类
- file-organizer  — 文件智能整理，批量重命名
- git-workflow    — Git 分支策略，提交规范
- code-review-quality — 自动化代码审查，风格检查
- workflow-automation — 工作流自动化，定时调度
- docker          — Docker 容器化，安全沙箱
- mysql           — MySQL 数据库查询，数据导出
- mcp-security-audit — MCP 安全审计，权限分析
- api-integration-specialist — REST API 集成，OAuth
- skill-vetter    — 技能质量审查

### 数据查询类
- weather         — 天气查询，无需 API
- baidu_search    — 百度搜索集成
- taobao-shop-price — 淘宝商品价格查询
- maishou         — 全网比价（淘宝/京东/拼多多/抖音/1688）

### 自动化/Agent类
- self_improving_agent — 自我改进 Agent
- Agent_Browser   — 浏览器自动化 Agent
- Find_Skills_Skill — 技能发现与推荐

### 通讯类
- feishu          — 飞书集成
- discord         — Discord 集成
- telegram        — Telegram 集成
- xiaohongshu-cli — 小红书 CLI
- twitter-media   — Twitter/X 媒体提取

## 第3步：安装飞书妙搭技能包（9项，需飞书 API）
- feishu-bitable       — 飞书多维表格
- feishu-calendar      — 飞书日历
- feishu-channel-rules — 飞书频道规则
- feishu-create-doc    — 创建飞书文档
- feishu-fetch-doc     — 获取飞书文档
- feishu-im-read       — 飞书消息已读
- feishu-task          — 飞书任务
- feishu-troubleshoot  — 飞书问题排查
- feishu-update-doc    — 更新飞书文档

## 第4步：安装 Miaoda 通用技能（12项）
- miaoda-coding           — AI 编程助手
- miaoda-database-skill   — 数据库操作
- miaoda-doc-parse        — 文档解析
- miaoda-image-understanding — 图片理解
- miaoda-openclaw-guide   — OpenClaw 指南
- miaoda-skillhub         — 技能中心

- miaoda-web-search       — 网页搜索
