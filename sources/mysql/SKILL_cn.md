> **来源：** planetscale/database-skills@mysql

## 何时使用

✅ **使用此技能的场景：**

- 分析数据库结构和关系
- 将查询结果导出为 CSV/JSON
- 编写和优化 SQL 查询
- 理解表结构和索引
- 数据库环境间迁移

## 快速开始

```bash
# 连接本地 MySQL
mysql -u root -p

# 列出数据库
mysql> SHOW DATABASES;

# 选择数据库
mysql> USE mydb;

# 列出表
mysql> SHOW TABLES;
```
