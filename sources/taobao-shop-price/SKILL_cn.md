# 全网电商比价

> **来源：** 飞书机器人技能库  
> **API 依赖：** 是 | **协议：** MIT

## 何时使用

✅ **使用此技能的场景：**

- 需要跨平台比价
- 需要找最低价
- 需要搜索淘宝/京东/拼多多商品
- 需要找优惠和优惠券
- 需要获取购买链接
- 用户提到"比价"、"哪里买便宜"、"价格对比"

❌ **不要使用的场景：**

- 需要购买虚拟商品
- 需要实时股票/汇率等金融数据

## 工作流程

### 第一步：检索商品

```bash
python3 scripts/price.py search --keyword "关键字" --sourceType 0 --pages 1 --format csv
```

### 第二步：过滤并输出推荐表格

- 保留名称与搜索商品相关的条目
- 按推荐度从高到低排序
- 推荐度综合考虑：相关性、价格、月销量

### 第三步：等待用户选品

输出检索概况和推荐商品，等待用户输入序号

### 第四步：获取购买链接

```bash
python3 scripts/price.py link --goodsId "<id>" --sourceType <type>
```

### 第五步：汇总输出

按价格升序输出最终表格和链接

## 使用示例

### 比价 iPhone

```
帮我比价 iPhone 16 Pro
```

### 指定平台搜索

```
帮我从京东检索 AirPods Pro
```

### 找最低价

```
帮我找最便宜的纸尿裤
```

## 输出表格格式

| 列 | 说明 |
|----|------|
| ID | 序号（用户选品用） |
| 名称 | 商品标题 |
| 平台 | 淘宝/京东/拼多多等 |
| 商铺 | 店铺名称 |
| 价格 | 现价 |
| 原价 | 标价 |
| 优惠 | 优惠券金额 |
| 月销 | 月销量 |
| 推荐度 | ⭐ 1-5 星 |
| 推荐理由 | 一句话说明 |

## 注意事项

- 价格为检索时点价格，可能变动
- 部分商品有优惠券，下单前确认
- 月销数据仅供参考
