📄 **Programmatic PDF Document Creation & Template Filling Tool**. ### [local-ocr](sources/local-ocr/SKILL_en.md)
🔍 **Local OCR Image-to-Text** based on Tesseract engine for offline text recognition without network API. Supports Simplified/Traditional Chinese, English, Japanese, Korean. Privacy-friendly and works offline. **Your system already has complete environment (Tesseract 5.5.2 + chi_sim/chi_tra/eng/jpn/kor language packs).**

[View Details](sources/local-ocr/SKILL_en.md)

**🚀 Start Immediately (Zero Configuration):**
```bash
# Enter skill directory and run test script
cd ai-agent-commonly-used---kills/sources/local-ocr
python3 ocr_test.py

# Recognize specific image
tesseract your_image.png stdout -l chi_sim --psm 6
```

**✨ Core Capabilities:**

- ✅ **Fully Offline**: No network, no API keys, no privacy leak risks
- ✅ **Multi-language Support**: Simplified/Traditional Chinese, Japanese, Korean + 80+ languages
- ✅ **Batch Processing**: Process all images in folder with one command
- ✅ **Python Automation**: Ready-to-use Python functions for direct calls
